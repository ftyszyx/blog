yml总体结构：
Config struct {
	Cache     Stringorslice
	Platform  string
	Branches  Constraint
	Workspace Workspace   工作目录，
	Clone     Containers  clone 配置
	Pipeline  Containers  a list steps to build
	Services  Containers
	Networks  Networks
	Volumes   Volumes
	Labels    libcompose.SliceorMap
}


1、registry 是用来获取私有docker境像用的
--repository <repository> \
  --hostname <image> \
  --username <name> \
  --password <value>

 Image gcr.io/foo/bar has hostname gcr.io
Image foo/bar has hostname docker.io
Image qux.com:8000/foo/bar has hostname qux.com:8000

存在drone数据库中

2、secrets用法
存一些密码相关信息在yml配置文件外，然后yml中可以直接使用
drone secret add \
  --repository <registry> \
  --image <image> \
  --name <name> \
  --value <value>
